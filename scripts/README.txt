README - Zenodo Metadata Prep

This project folder includes:
- pdf/: place your PDF files here
- metadata/: generated JSON metadata files for Zenodo
- source_tables/: contains metadata_input.csv with details for batch files
- scripts/: your Python scripts to convert CSV to JSON or upload to Zenodo

Next steps:
1. Edit metadata_input.csv to reflect your PDF files.
2. Use a Python script to convert CSV rows to JSON.
3. Upload to Zenodo manually or with automation.
